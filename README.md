# Harshith Battu - 3D Creative Portfolio

Welcome to my interactive 3D portfolio! This project showcases a fully immersive **Cyberpunk Hologram Table** built with React, WebGL, and Three.js, serving as the backdrop for my professional experience, skills, and featured projects.

👉 **[View the Live Portfolio Here](https://h4026492-hash.github.io/portfolio/)**

## ✨ Features
- **Cyberpunk Hologram Table**: A fully interactive 3D background featuring spinning cosmic rings, a pulsing central core, and floating 3D navigation elements.
- **Scroll Syncing**: The 3D world is seamlessly synchronized with the smooth-scrolling HTML interface.
- **Glassmorphism UI**: Beautiful, semi-transparent overlays to showcase my modern design aesthetics.
- **Dynamic Animations**: Powered by Framer Motion for buttery-smooth layout transitions and stagger effects.
- **Contact Integration**: Live, functioning form submissions utilizing FormSubmit.co.

## 💻 Tech Stack
- **Framework:** React 19 + Vite
- **3D Graphics:** Three.js + React Three Fiber (`@react-three/fiber`)
- **3D Utilities:** Drei (`@react-three/drei`)
- **Styling:** Tailwind CSS V4
- **Animations:** Framer Motion

## 🚀 Running Locally

If you'd like to clone and run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/h4026492-hash/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

## 📦 Deployment
This project is automatically built as a high-performance single-page file and distributed via GitHub Pages.
- To deploy updates from local: `npm run deploy`
